# AdminBarber

Sistema de gestão completo para barbearias, focado em agendamento online, controle financeiro e fidelização de clientes.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Como rodar localmente

1. Instale as dependências:
   ```sh
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

3. Para build de produção:
   ```sh
   npm run build
   ```

## PWA

O sistema está configurado como um Progressive Web App (PWA), permitindo a instalação no celular e funcionamento offline básico.
